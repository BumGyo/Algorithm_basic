  b. 전위 탐색 알고리즘

2. 넓이 우선 탐색 알고리즘
  -> 레벨 오더 탐색 알고리즘 : 
     루트 노드로 시작해서 바로 아래에 있는
     모든 자식 노드들을 왼쪽에서 오른쪽 방향으로 방문(탐색)

1. 깊이 우선 탐색 알고리즘
  a. 중위 탐색 알고리즘 : In-order
     : 기준 루트(root) 노드 : 최상위 노드 : 첫 번째 노드
     : 첫 번째로는 root 노드의 왼쪽 자식 노드
     : 모든 왼쪽 자식 노드들을 방문한 후에는
       루트 노드를 방문
     : 마지막으로 남아있는 오른쪽 자식 노드들을 방문

왼쪽 자식 노드 -> 루트 노드 -> 오른쪽 자식 노드

  b. 전위 탐색 알고리즘 : Pre-order
     -> 제일 먼저 루트 노드를 방문
     -> 왼쪽 자식 노드들을 방문
     -> 오른쪽 자식 노드들을 방문

  c. 후위 탐색 알고리즘 : Post-order
     -> 제일 나중에 루트 노드를 방문
     -> 왼쪽 자식 노드 -> 오른쪽 자식 노드 -> 루트 노드
