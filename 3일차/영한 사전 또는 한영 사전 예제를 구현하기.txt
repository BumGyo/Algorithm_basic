* 영한 사전 또는 한영 사전 예제를 구현하기

1. 영한 사전 : 키를 영어로 설정
예) 사용자가 apple을 검색한 경우에는 화면에
    출력되는 결과는 사과
1) 새로운 변수를 선언 : 
    my_eh_dict = {}
2) 사용자로부터 키와 값을 입력 받은 다음에
   my_eh_dict 사전 변수에 저장합니다!!
가. 사용자가 입력한 키를 보관할 변수를 선언
    : 
key1 = 
  input("사전에 저장할 키를 입력하세요: ", 
         end="")
나. 사용자가 입력한 값을 보관할 변수를 선언
value1 = 
  input("영한 사전에 저장할 값을 입력하세요: ", 
  end="")
다. my_eh_dict[key1] = value1

2. 한영 사전 : 키를 한글로 설정
