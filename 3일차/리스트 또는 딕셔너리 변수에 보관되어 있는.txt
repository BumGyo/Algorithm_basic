* 리스트 또는 딕셔너리 변수에 보관되어 있는
  데이터들을 순서에 맞게 재배열하는 방법
-> 정렬(Sorting)

* 정렬 알고리즘의 종류
1) 순차(선택) 정렬 알고리즘
2) 버블(Bubble: 거품)  정렬 알고리즘
3) 삽입(Insertion) 정렬 알고리즘
4) 병합(Merge) 정렬 알고리즘
5) 퀵(Quick) 정렬 알고리즘

1) 순차(선택: Selection) 정렬 알고리즘
   -> 첫 번째 데이터를 선택(기준)
     -> 나머지 데이터들과 비교
예) list1 = [5, 10, 6, 99, 300]
5 가 먼저 선택 
   (10,6,99,300 데이터들과 차례대로 비교)
-> 오름차순 정렬 : 제일 작은 값을 찾기
    -> 기준 데이터 보다 더 작은 값을
       찾은 경우에만 데이터의 위치를 서로 바꿈
       -> 스왑 알고리즘을 사용
* 스왑 알고리즘 개요
-> 데이터를 갖고 있지 않은 임시 변수 temp를
   사용
temp = 기준 데이터 또는 비교 데이터
기준 데이터를 보관하고 있던 변수이름 =
    비교 데이터
비교 데이터를 보관하고 있던 변수이름 = temp
-> 파이썬에서는 임시 변수를 사용하지 않고도
   데이터의 위치를 교환 가능
리스트변수이름[기준데이터의위치],
  리스트변수이름[비교데이터의위치]
   = 
리스트변수이름[비교데이터의위치],
  리스트변수이름[기준데이터의위치]

* 리스트 변수가 갖고 있는 sort( ) 함수를 사용
-> 리스트변수이름.sort( )
