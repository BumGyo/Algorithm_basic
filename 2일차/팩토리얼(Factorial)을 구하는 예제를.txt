* 팩토리얼(Factorial)을 구하는 예제를
  반복문을 사용해서 구현하기

1) 팩토리얼 : !(느낌표) 연산자를 사용
   -> 1 ~ n 까지의 곱셈
예) 3! -> (3-1)! * 3 -> 2! * 3
       -> (2-1)! * 3 * 2
       -> 1! * 3 * 2
       -> 3 * 2 * 1 -> 결과는 6

2) for 반복문을 사용해서 3! 팩토리얼을
   계산한 후에 화면에 결과를 출력
-> 곱셈 결과를 보관할 변수가 필요합니다!!
   -> result = 1
for i in range(1, 4):
   result = result * i
