1. 10진수를 2진수로 바꾸는 알고리즘

0. 진수 : Radix

1) 10진수 : 0 ~ 9
  용어 : 가중치(weight) : 자릿값
예) 100 : 1 * 10의 2승 + 0 * 10의 1승 +
          0 * 10의 0승
2) 2진수 : 컴퓨터 : 0,1
예) 0, 1, 00, 01, 10, 11, ....
3) 8진수 : 0~7
4) 16진수 : 
   0~9, A(10),B(11),C(12),D(13),E(14),F(15)
예) A (이름 : 식별자 : 변수 또는 함수 또는
    클래스)
-> 소문자 x 또는 대문자 X를 사용하면
   16진수를 표현 : 0xA 또는 0XA
