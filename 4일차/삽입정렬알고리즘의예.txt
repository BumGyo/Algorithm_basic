* 삽입 정렬 알고리즘의 예)

1. 시뮬레이션해 보기

1) 가상의 데이터를 준비 :
   list1 = [30, 20, 50, 10, 70]
변수 i와 j의 변화되는 패턴의 이해
변수 i는 1(두 번째 데이터의 위치)
변수 j는 변수 i와 같게
변수 value = 두 번째 데이터 : 20

가. 처음에는 첫 번째 데이터인 30과 20을 비교
-> 리스트변수이름[j-1] > value
   -> 현재 리스트변수이름[j-1]에 있는
      데이터가 j 번째 위치로 이동
      : 리스트변수이름[j] = 리스트변수이름[j-1]

-> 현재 두 번째 데이터는 변수 value에 
   있는 상태 : 
   list1 = [30, 30, 50, 10, 70]

-> 변수 j의 위치를 하나 감소
