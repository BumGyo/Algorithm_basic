1. 연결 리스트(Linked List)

1) 종류
  가. 단순 연결 리스트 : 
       다음 노드의 주소(위치)를 보관하는 변수를 하나만 준비
예) # 클래스(Class)로 정의
class MyNodeClass:
    # 초기화 함수(멤버 변수에 초기값:시작값을 정해주는 함수)
    def __init__(self, value_p, next_p = None)
-> 변수이름 = 클래스이름( 데이터 )
-> 또는
   변수이름 = 클래스이름 ( 데이터, 주소 )
# None : 아직 메모리 공간의 주소가 할당되지 않은 상태 값

    # 콜백 함수(Call-back Function) : 함수의 정보를
    # 파이썬 프로그램에서 알고 있는 함수 : 자동 실행되는 함수
    # 예) 변수이름 = 클래스이름( )
* 매개 변수 이름의 끝에는 접미사 _p를 붙이도록 하겠습니다.
def __init__(self, value_p, next_p)
# 변수이름 = 클래스이름( 데이터, 주소 )
